# 🏥 FitTrack Analytics

A full-stack health and fitness analytics web application built with Python (Flask) and SQLite. This app allows users to log their daily fitness data and track their health goals over time.

---

## 🚀 Features

- Add and manage user profiles
- Log daily health data (steps, calories, sleep, heart rate, water intake)
- Set and track fitness goals
- RESTful API built with Flask
- Lightweight SQLite database

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Backend | Python, Flask |
| Database | SQLite, SQLAlchemy |
| API Testing | Thunder Client |
| Version Control | Git & GitHub |
